Консольное приложение с двумя методами сортировки.
1. void QuickSort(int[] array) сортирует массив целых чисел по алгоритму "быстрой сортировки".
2. void MergeSort(int[] array) сортирует массив целых чисел по алгоритму "сортировка слиянием".

Тестирование производится на двух случайных массивах Random(100);
